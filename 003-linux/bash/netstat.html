安装

使用

netstat -apn

p代表显示进程的名称和pid a代表以数字形式展示，因为有时dns查询比较耗时。

a表示所有socket类型，其他可选值：

 <Socket>={-t|--tcp} {-u|--udp} {-U|--udplite} {-S|--sctp} {-w|--raw}
       {-x|--unix} --ax25 --ipx --netrom

简单来说还可以用：t,u,x

显示监听列表

netstat -l

# 或

netstat --listening

显示路由表

netstat -r

# 或

netstat --route

显示网络接口

netstat -i
netstat --interfaces

显示效果类似于：
